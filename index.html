📖 Typeit 기계식 직타식 타자기 가이드

1. 실물 먹지 활판 원리 고증 (Direct Stamping)

글쇠를 치는 즉시 종이에 먹색 잉크 도장이 고스란히 내려앉아 굳어지는 아날로그 스탬핑 방식을 재현했습니다.

2. 한글 타자법 (초성 / 중성 / 받침 결합 메커니즘)

  • 자음(초성): 타격 시 캐리지가 움직이지 않고 제자리(X)를 유지합니다.
  • 모음(중성): 타격 완료 시 우측으로 자간(9.5px)만큼 1칸 전진합니다.
  • 받침(종성): Shift 키(PC 물리 자판) 또는 화면의 '받침' 키를 활성화하고 자음을 치면 이전 글자 밑 받침단에 정확히 잉크를 융합시킵니다.

💡 예: "하" ➔ [ㅎ] ➔ [ㅏ] / "한" ➔ [ㅎ] ➔ [ㅏ] ➔ [Shift 누른 채 ㄴ]

3. 오타 정정 (Lift-off 백스페이스)

백스페이스를 누르면 마지막에 종이에 찍혔던 글자 잉크를 리프트오프 교정 테이프로 뜯어내며 캐리지를 전 단계로 물리적으로 역송시킵니다.

✂️ 타자기에서 뜯어낸 소중한 기록

조선100년 기념체로 정교하게 각인된 A4 규격의 리얼 활판 인쇄지가 제작되었습니다.
